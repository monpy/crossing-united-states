# 出発：自宅 → 羽田/成田 → サンフランシスコ

> DAY 1（10/9 金）・有給ゼロで“まる1日”得するスタート

## 道筋
1. **10/9（金）日中**：通常通り出勤
2. 仕事後 → 羽田（HND）または成田（NRT）へ（大阪からは夕方の便で東京入り）
3. **夜便でSFOへ出発**（SFO直行便あり）
4. 飛行 約9.5h。日付変更線を越えるので **同じ金曜の昼過ぎ** にSFO着
5. SFO → 市内（BART で約30分・$10前後）

## ポイント
- 金曜を休まないので **有給を1日も使わずに初日を稼げる**（夜発・同日着のマジック）。
- 着いた日は時差調整がてら軽く街歩き程度に。
- レンタカーは土日のうちに受け取り（市内は車不要なので受取は出発前日でOK）。

## 持ち物リマインド
- パスポート / ESTA（事前申請）
- 国際運転免許証 ＋ 日本の免許証
- 機内泊用の上着・アイマスク
