# 🎒 必要なもの（チェックリスト）

## 出発前（必須）
- [ ] **ESTA**（電子渡航認証）※日本のパスポートなら申請。$21・有効2年
- [ ] **国外運転免許証（IDP）** ＋ 日本の運転免許証（両方携帯。IDP単体は無効）
- [ ] **クレジットカード**（レンタカーのデポジット用。複数枚・各自持参）
- [ ] **海外旅行保険**（治療費無制限プラン推奨。アメリカは医療費が超高額）
- [ ] **航空券**（SF in / NY out のオープンジョー）
- [ ] パスポート残存期間の確認（半年以上推奨）

## 車・運転まわり
- [ ] レンタカーの **追加賠償責任（LIS/SLI）** ＋ **LDW/CDW**（[車のページ](#/car)参照）
- [ ] **運転する全員を追加ドライバー登録**
- [ ] **Googleマップのオフライン地図**（国立公園・砂漠は電波が来ない）
- [ ] 車載スマホホルダー、USB充電器（シガーソケット）、GasBuddyアプリ

## キャンプ・車中泊（節約の要）
- [ ] テント or 車中泊用マット・寝袋（夜は冷える）
- [ ] **コンロ（小型ガスバーナー）＋ クッカー**（自炊で食費を圧縮）
- [ ] **大きめの水ジャグ＋各自の水筒**、クーラーボックス
- [ ] ヘッドランプ／ランタン（星空観賞にも）
- [ ] 簡単な食器・カトラリー、ゴミ袋

## 自然エリア装備
- [ ] **重ね着できる服**（砂漠は日中35℃・夜10℃、山は寒い）
- [ ] トレッキングシューズ、帽子、サングラス、**日焼け止め**
- [ ] モバイルバッテリー、常備薬、虫除け、救急セット

## 通信・お金
- [ ] **eSIM or 米国SIM**（データ多めプラン）
- [ ] 現金少々（チップ・現金のみの田舎の店用）＋ タッチ決済カード

## あると快適
- [ ] America the Beautiful パス（公園入口でも購入可）
- [ ] 長距離用プレイリスト／オーディオブック
- [ ] スーパーでの食料調達計画
