# 概要

> コンセプト：**14日間・できるだけ貧乏旅行**で、西海岸→東海岸を車で横断。
> 好み（自然 / その土地でしかないイベント / 美術館）に最適化した男2〜3人の旅。

## 結論サマリー（14日・節約版）

| 項目 | 内容 |
|------|------|
| 期間 | **14日間（13泊）** |
| ペース | **ハード**（総走行 約5,400km / 3,400マイル）。自然メインで効率重視 |
| ベスト時期 | **9月中旬〜10月上旬**（砂漠が涼しい・Tioga Pass開通・紅葉） |
| ルート | サンフランシスコ発 → 南西部の国立公園群 → デンバー → 平原横断 → シカゴ → クリーブランド → ニューヨーク着 |
| 車 | **コンパクトSUV**（節約と荷物のバランス。座席を倒して車中泊も可） |
| 宿 | **野営（キャンプ）＋ホステル＋安モーテル**を組み合わせて節約 |
| 予算 | **1人あたり 約 23〜30万円**（国際線除く・3人割り勘・節約前提） |

> ⚠️ **ペース注意**：14日だと平原（デンバー以東）と東部で長時間運転の日が出ます。
> もっとゆったりしたい場合は [🗺️ ルート](#/route) の「ペース調整案」を参照。
