# 💰 費用（貧乏旅行・節約版）

> 14日間 / 男3人で割り勘 / 野営＋自炊前提の概算。1人あたり。

| 項目 | 概算（1人） | 節約の前提 |
|------|------------|-----------|
| レンタカー（コンパクトSUV・14日・乗り捨て込） | **¥6〜8万** | 3人割り。早期予約・空港外店舗 |
| ガソリン | **¥2〜3万** | 5,400km ÷ 28mpg。3人割り |
| 宿泊（13泊） | **¥5〜6万** | 野営＋ホステル＋安モーテルのミックス。たまに車中泊 |
| 食事 | **¥5〜7万** | スーパー調達＋キャンプ自炊中心。外食は最小限 |
| 国立公園パス | **¥0.4万** | America the Beautiful 年間パス $80 を3人で割る（1台分） |
| アクティビティ・美術館 | **¥1〜1.5万** | 自然は基本無料。シカゴ美術館 $32 等。有料ツアーは省く |
| 海外旅行保険 | **¥1.5万** | 必須。医療費が高額なので削らない |
| **小計（国際線除く）** | **約 ¥23〜30万 / 人** | |
| 国際線（SF in / NY out オープンジョー） | **¥15〜30万** | 時期・予約次第 |

> 💰 **総額の目安：国際線込みで 1人 約 40〜55万円**。

## 節約術まとめ
- **宿**：国立公園のキャンプ場（$20〜35/泊を割り勘）、ホステル、Motel 6など格安チェーン。車中泊も活用。
- **食事**：スーパー（Walmart/Trader Joe's）でまとめ買い→キャンプ自炊。サンドイッチ・シリアル・パスタ。
- **水**：大きいジャグで買い、各自の水筒に。砂漠は脱水注意。
- **公園**：年間パス $80 が単発より圧倒的に得（1台＝同乗者カバー）。
- **美術館**：**スミソニアン（D.C.）は無料**。各館の「無料デー」を狙う。
- **ガソリン**：GasBuddyアプリで安いスタンドを探す。州により価格差大。
- **長距離ワープ**：平原は片道格安航空券に切り替えるとレンタル代・ガソリン・宿が浮くことも（[ルートのペース調整案](#/route)参照）。
